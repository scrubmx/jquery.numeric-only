# jQuery Numeric Only

[![Code Climate](https://codeclimate.com/github/scrubmx/jquery.numeric-only/badges/gpa.svg)](https://codeclimate.com/github/scrubmx/jquery.numeric-only)
[![License](https://poser.pugx.org/pugx/badge-poser/license.svg)](https://github.com/scrubmx/jquery.numeric-only/blob/master/licence.txt)

Limit the values for any input to numeric values only.

## Usage
```js
$('#phone').numericOnly();
```
